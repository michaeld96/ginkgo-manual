1 Projects Page

Click on Projects in the navagation bar at the top of the screen will bring us to the projects page. This page is where users will create new projects, edit existing projects, and delete projects.

Picture of the Projects Page

1.1 Creating a Project

Click the New Project button to open a project creation interface. When the New Project window pops up, this is where we will load in our data. When uploading the data by clicking on Upload Data the title of the project will automatically be named the file if not specified.

New Project Page

1.1.1 Upload Data

Data must be uploaded as an .xslx file. Please reference the Data Specification section of the IQnca documentation to find which columns are required for analysis(https://iqnca.intiquan.com/). Data in columns can be named before importing data, or can be adjusted in the app at a later time.

1.1.2 Project Tags

Give an appropriate name and description to the project.

Click Create New Project to initialize and open the new project.

1.2 Opening A Project

Double click on a project to open it. Note If this is a new project, clicking Create New Project will bring the user into the main landing page of the application.

1.3 Deleting Data

To delete projects, single-click the project, and then click the bin icon. A prompt will ask to confirm this action.

1.4 DATA IS NOT SAVED

Persistent data storage is a feature that is currently being worked on. Projects are currently tied to a session.
